# AI Tool Selection

## Decision Question
Which AI-assisted development tool should be used during the course?

## Context Snapshot
The course includes AI-assisted software development and modern engineering practices.

## Alternatives Considered
- GitHub Copilot
- Cursor
- ChatGPT
- Manual development only

## Decision
Use GitHub Copilot as the primary development assistant.

## Reasoning
Copilot is directly integrated into the repository workflow and simplifies experimentation.

## Expected Outcomes
- Faster documentation creation.
- Improved productivity.
- Better GitHub integration.

## Reconsideration Conditions
If another tool demonstrates significantly better results.
