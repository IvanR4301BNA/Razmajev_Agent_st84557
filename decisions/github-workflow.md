# GitHub Workflow

## Decision Question
How should the project and learning progress be tracked?

## Context Snapshot
The course requires public GitHub repositories and emphasizes version history, commits, and transparent progress.

## Alternatives Considered
- Local folders
- Google Drive
- GitHub repository

## Decision
Use GitHub with Markdown files and regular commits.

## Reasoning
GitHub provides version control, public visibility, commit history, and a clear structure for tracking learning artifacts.

## Expected Outcomes
- Clear evidence of progress.
- Easier review by the instructor.
- Better organization of beliefs, decisions, experiments, and synthesis.

## Reconsideration Conditions
This decision should be reconsidered only if the course requirements change.
