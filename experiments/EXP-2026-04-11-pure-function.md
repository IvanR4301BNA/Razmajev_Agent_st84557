
# Experiment: Pure Function Login

## Prompt Used

Write login logic as a pure function using the provided requirements and Mermaid diagram. The function must have no side effects and return predictable output.

## Result

The AI generated a function that processes input and returns a result without using external state.

## Evaluation

The first result was partially correct. After refining the prompt, the AI produced a proper pure function.

## Conclusion

BDD and Mermaid diagrams improved the AI output. The pure function constraint ensured predictable behavior.
